# LAPSIC | Espaço do Ligante

Site da Liga Acadêmica de Psicologia Clínica (LAPSIC) — Dashboard do Ligante com cronograma, diretoria e mural de avisos.

## Stack

- React 19 + Vite
- Lucide Icons
- Deploy: Vercel

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
